正确答案是 **B:列表是可变长度的,可以追加和删除元素**。

逐项分析:

- ❌ A "长度不可变"——错。list 创建后可用 `append` / `extend` / `pop` 改变长度,这正是 list 与 tuple 的关键差异
- ✅ B "可变长度"——对
- ❌ C "所有元素必须同类型"——错。list 是**异质**的,`[1, "two", 3.0]` 完全合法
- ❌ D "索引从 1 开始"——错。Python 索引从 **0** 开始,`a[0]` 才是首元素

如果你以前写过 C / Java 的定长数组,选项 A 和 C 是常见迁移误区——这两个错误观念会一直影响你后面学切片和 numpy。
