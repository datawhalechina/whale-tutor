# 列表的创建与表示

Python 的 **list** 是一个**有序、可变长、异质**的容器。

## 创建方式

最常见的两种:

- **字面量语法** `[]`: 写法 `a = []`、`b = [1, 2, 3]`
- **构造函数** `list()`: 写法 `c = list()`(空)、`d = list("abc")`(从可迭代对象转换得到 `['a', 'b', 'c']`)

## 关键属性

- **有序**: 每个元素有固定位置,索引从 0 开始,`a[0]` 是首元素
- **可变长**: 创建后可用 `append` 追加、`pop` 删除,长度随时变化
- **异质**: 不像 C 数组要求同类型,list 里可以混放 int / str / float / bool / 甚至另一个 list
- **求长度**: `len(a)` 返回元素个数,空 list `len([])` 是 0

## 一个最小例子

```python
a = []                       # 空列表
a.append(1)                  # 现在 a == [1]
b = [1, "two", 3.0, True]    # 异质元素都合法
len(b)                       # 4
```

## 为什么这么设计

list 是 Python 提供的"通用 sequence"——它不追求极致性能(那是 NumPy 的事),而是追求灵活性和易用性。理解这个设计取舍,能帮你后面学到 tuple、array、numpy.ndarray 时知道各自的适用场景。
