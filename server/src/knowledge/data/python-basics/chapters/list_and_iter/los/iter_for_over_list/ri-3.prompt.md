## 筛选偶数

写一个函数 `even_only(nums)`,返回 `nums` 中**所有偶数**组成的**新列表**。

要求:
- 用 `for` 循环 + `append` 实现
- **不修改原列表**(传入的 nums 不变)
- 偶数判定用 `x % 2 == 0`
- 保持原顺序

例如:

```python
even_only([1, 2, 3, 4, 5])    # → [2, 4]
even_only([2, 4, 6])          # → [2, 4, 6]
even_only([])                 # → []
```

提示:在函数开头建一个 `result = []`,逐个检查 `nums` 中的元素决定是否 `append`。
