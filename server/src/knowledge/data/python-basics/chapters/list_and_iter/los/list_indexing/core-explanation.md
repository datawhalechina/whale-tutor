# 索引与切片

列表是**位置容器**,所以你需要一种方式按位置取元素或子序列——这就是索引和切片。

## 索引(单个元素)

```python
a = [10, 20, 30, 40, 50]
a[0]    # 10  (首)
a[2]    # 30
a[-1]   # 50  (尾,负索引从右往左数)
a[-2]   # 40
a[10]   # IndexError: list index out of range
```

**关键**:
- 索引从 **0** 开始,不是 1
- 负索引: `a[-1]` 是最后一个,`a[-2]` 倒数第二
- 越界 → `IndexError`(运行时崩)

## 切片(子序列)

```python
a[i:j]      # 从 i 开始,到 j 之前(左闭右开)
a[i:j:k]    # 步长为 k
a[:]        # 完整拷贝
a[::-1]     # 反转
```

**关键**:
- `a[1:3]` 取 index 1, 2 共 **2 个**元素,**不包括 3**
- 越界切片**不报错**,自动截断:`a[5:10]` 在长度为 3 的 list 上返回 `[]`
- 这与索引越界(IndexError)是**两套行为**

## 一个对照例子

```python
a = [10, 20, 30]
a[1]      # 20
a[1:2]    # [20]      ← 切片返回 list,即使只有一个元素
a[5]      # IndexError
a[5:10]   # []        ← 切片越界返回空 list
a[-1]     # 30
a[::-1]   # [30, 20, 10]
```

理解了切片的"左闭右开"和"越界静默",你就能预测大多数列表表达式的输出。
