正确答案是 **`[1, 2, [3, 4]]`**。

`append(x)` 不管 x 是什么类型,都把它**作为一个完整对象**加到末尾。所以 `[3, 4]` 这个 list 整体进去,变成 `[1, 2, [3, 4]]` — 一个 3 元素的嵌套列表。

如果你想要 `[1, 2, 3, 4]`(展开后追加),要用 `extend`:

```python
a = [1, 2]
a.extend([3, 4])    # [1, 2, 3, 4]
# 或
a += [3, 4]         # 等价于 extend
```

记忆口诀:
- **append**: 加 1 个元素(可以是任何对象,包括 list)
- **extend**: 加多个元素(从可迭代对象展开)

什么时候你会真的想要 `[1, 2, [3, 4]]` 这种嵌套?在表示二维数据时(矩阵的行/列),嵌套是合理的。

什么时候是 bug?当你以为是拼接但写成了 append,得到嵌套后续操作就会全错。
