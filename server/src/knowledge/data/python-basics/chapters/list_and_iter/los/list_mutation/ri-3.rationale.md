正确答案是 **`[10, 20]`**。

逐步分析:

1. `a = [10, 20, 30]` — `a` 指向一个 list 对象
2. `b = a` — `b` 也指向**同一个**对象。`a` 和 `b` 是一对别名,**没有产生新对象**
3. `b.pop()` — pop 修改的是那个共享对象,弹出末尾的 30
4. 因为 `a` 和 `b` 指同一对象,`a` 现在也是 `[10, 20]`

如果你期望 `a` 保持 `[10, 20, 30]` 不变,在 step 2 应该显式**拷贝**:

```python
b = a[:]        # 浅拷贝(切片技巧)
b = list(a)     # 等价
b = a.copy()    # 等价
```

之后 `b.pop()` 只修改 `b`,`a` 仍是 `[10, 20, 30]`。

## 怎么验证我有没有搞混

如果你想确认两个变量是不是别名,用 `is`:

```python
a = [1, 2, 3]
b = a
print(a is b)   # True   ← 别名
b = a[:]
print(a is b)   # False  ← 不同对象,即使内容相同
print(a == b)   # True   ← 内容相等
```

`is` 比较**对象身份**(同一个对象?),`==` 比较**值相等**。这是 Python 中最重要的一对操作符。
