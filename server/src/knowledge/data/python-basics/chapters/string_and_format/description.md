**字符串(str)** 是 Python 中最常用的不可变序列。从读文件到拼日志、解析数据,几乎每段业务代码都在跟 str 打交道。

本章 2 个学习目标(LO),从字符串的本质讲起,过渡到日常会用到的方法 + f-string 格式化。

学完这一章,你能够:

- 区分字符串字面量的 3 种写法(单引号 / 双引号 / 三引号),知道反斜杠转义和 raw string
- 理解字符串**不可变性**(`s += 'x'` 实际上发生了什么)
- 用 `.split` / `.strip` / `.replace` / `.join` 等核心方法做常见处理
- 用 f-string 拼字符串(替代 % 和 `.format`)
